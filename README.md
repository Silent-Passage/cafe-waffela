# 🧇 Café Waffela – Digital Experience
> **Vom Duft frisch gebackener Waffeln direkt in den Browser**

[![Status](https://img.shields.io/badge/Status-Live-success?style=for-the-badge)]()
[![Location](https://img.shields.io/badge/Location-Lustenau,%20AT-orange?style=for-the-badge)](https://maps.google.com/?q=Cafe+Waffela+Lustenau)

Dieses Repository enthält den Quellcode für die offizielle, hochkonvertierende Website von **Café Waffela**. Ziel der Seite ist es, die 4,8-Sterne-Qualität des Cafés digital erlebbar zu machen und Laufkundschaft in Lustenau zu treiben

---

## ✨ Features & Highlights

### 🧇 Visuelles Menü
Ein interaktives Menü, das zwischen **Sweet (Waffeln & Softeis)** und **Savory (Bagels & Bowls)** unterscheidet. Inklusive "Best Seller" Badges basierend auf echten Kundenrezensionen

### ⭐ Social Proof Integration
- **Live Google Reviews:** Automatische Anzeige der neuesten 5-Sterne-Bewertungen
- **Instagram Feed:** Ein visuelles Schaufenster (@CafeWaffela), das die Ästhetik des Ladens zeigt
- **UGC (User Generated Content):** Sektion für Kundenfotos mit dem Hashtag #CafeWaffela

### 🚀 Conversion-Optimiert
- **Mobile-First:** Perfekt bedienbar für Gäste, die von unterwegs suchen
- **One-Click-Navigation:** Direkte Verlinkung zu Google Maps ("Bring mich hin")
- **WhatsApp-Quick-Chat:** Für schnelle Anfragen zu Tagesangeboten oder Softeis-Verfügbarkeit

---

## 🎨 Branding & Vibe
- **Primärfarbe:** `#F4A460` (Waffelgold)
- **Akzentfarbe:** `#E91E63` (Beeren-Pink / Instagram Vibe)
- **Typography:** Weiche Serifen für Überschriften (fluffig) & minimalistische Sans-Serif für Lesbarkeit

## 🛠️ Tech Stack & Hosting

- **Frontend:** HTML5, Modern CSS (Flexbox/Grid), Vanilla JavaScript
- **APIs:** - Instagram Basic Display API (für den Feed)
  - Google Business Profile API (für Rezensionen)
- **Hosting:** Optimiert für Self Hosting oder Plattformen wie Netlify/Vercel
